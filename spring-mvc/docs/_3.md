# 동시 요청 - 멀티 쓰레드

## 쓰레드

- 애플리케이션 코드를 하나하나 순차적으로 실행하는 것은 쓰레드
- 자바 메인 메서드를 처음 실행
  '?하면 main이라는 이름의 쓰레드가 실행
- 쓰레드가 없다면 자바 애플리케이션 실행이 불가능
- 쓰레드는 한번에 하나의 코드 라인만 수행
- 동시 처리가 필요하면 쓰레드를 추가로 생성

> 단일 요청 - 쓰레드 하나 사용

> 다중 요청 - 쓰레드 하나 사용

> 요청마다 쓰레드 생성

- 특징

- 장점

- 단점

> 쓰레드 풀

- 요청 마다 쓰레드 생성의 단점을 보완
